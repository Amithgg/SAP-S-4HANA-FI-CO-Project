🔄 Project Workflow:

1️⃣ Introduction to SAP & S/4HANA:

- Overview of SAP ecosystem and modules
- Understanding SAP Easy Access screen and navigation
- Learning transaction codes (T-Codes) and menu paths

2️⃣ Organizational Structure Setup:

- Created **Company Code** using T-Code `OX02`
- Created **Chart of Accounts** using T-Code `OB13`
- Assigned Company Code to Chart of Accounts using `OB62`
- Configured Business Areas and Credit Control Areas

3️⃣ Financial Accounting (FI) Operations:

- Created **General Ledger (GL) Accounts** using `FS00`
- Posted **GL Account Documents** using `FB50`
- Created **Vendor Accounts** using `XK01`
- Entered **Vendor Invoices** using `FB60`
- Created **Customer Invoices** using `FB70`
- Processed **Incoming Payments** using `F-28`
- Processed **Outgoing Payments** using `F-53`

4️⃣ Controlling (CO) Operations:

- Created **Cost Centers** using `KS01`
- Created **Profit Centers** using `KE51`
- Created **Cost Elements** using `KA01`
- Entered Plan Data for Cost Centers using `KP06`
- Generated **Cost Center Reports** using `KSB1`

5️⃣ Integration & Reporting:

- Explored FI-MM (Materials Management) integration
- Explored FI-SD (Sales & Distribution) integration
- Generated financial reports and variance analysis
- Practiced SAP Fiori Apps for FI, MM, and SD

6️⃣ Assessment & Feedback:

- Completed Q&A and troubleshooting sessions
- Underwent final assessment and received performance feedback
- Obtained SAP S/4HANA Finance certification via SAP Learning Hub

|--------|-------------|

| `FS00` | Create / Display General Ledger Account |
| `FB50` | Post General Ledger Document |
| `FB60` | Enter Vendor Invoice |
| `FB70` | Enter Customer Invoice |

| `F-53` | Post Outgoing Payments |
| `FB03` | Display Document |
| `OB52` | Define Open and Closing Periods |
